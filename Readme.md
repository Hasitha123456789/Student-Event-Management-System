# 🎓 EventHub - Student Event Management System

A comprehensive event management system for universities, colleges, and schools with AI-powered features.

## ✨ Features

### For Students
- ✅ Browse and register for events
- ✅ View registered events with QR codes
- ✅ Download participation certificates
- ✅ Submit feedback and ratings
- ✅ Receive AI-powered event recommendations

### For Event Organizers
- ✅ Create and manage events
- ✅ Track registrations and attendance
- ✅ Manage event budgets and expenses
- ✅ Recruit and manage volunteers
- ✅ Generate reports and analytics

### For Administrators
- ✅ Approve/reject event proposals
- ✅ Oversee all campus events
- ✅ Generate university-wide reports
- ✅ Manage users and permissions

### 🤖 AI-Powered Features
- **Attendance Prediction** - Forecasts event attendance rates
- **Schedule Conflict Detection** - Prevents double-booking venues
- **Budget Recommendations** - Suggests optimal budget allocation
- **Smart Event Suggestions** - Personalized event recommendations
- **Natural Language Summaries** - Human-readable event status
